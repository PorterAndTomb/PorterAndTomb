PorterAndTomb

3D virtual world for collaborative programming

Download as .zip Download as .tar.gz View on GitHub

Welcome to PorterAndTomb.

PorterAndTomb is a 3-dimensional virtual world built for collaborative programming. We are trying to create an environment/experience that is similar to physically pair programming with other developers. Think of it like a virtual office. You can move around and look at what other people are working on, call someone over to help you, or you can just write code in a unique environment!

Ready to take the red pill?
